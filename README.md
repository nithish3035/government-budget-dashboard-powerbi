# 📊 Government Budget Dashboard (Power BI)

An end-to-end Power BI project analyzing Tamil Nadu's state government finances across FY 2020-25.

## 🎯 Project Overview
This dashboard tracks government budget allocation, expenditure, scheme performance, and fiscal health across departments and districts, helping identify overspending, underutilization, and budget variance trends.

## 🛠️ Tools Used
- **Excel** – initial data sourcing & structuring
- **Power Query** – data cleaning & transformation
- **DAX** – calculated measures & KPIs
- **Power BI** – data modeling & interactive dashboard design

## 📄 Dashboard Pages
- **Overview** – GSDP, fiscal deficit, budget utilization, funding source mix
- **Department** – budget vs. actual expenditure by department
- **Schemes** – utilization %, variance & underutilization ranking across 129+ schemes
- **Variance** – YoY budget growth, fiscal deficit trends, capital outlay vs. borrowings
- **Insights** – district-wise spend classification (On Track / Over Budget / Under Budget)

## 🔑 Key Features
- Interactive slicers by fiscal year, district, and purpose
- Drill-through analysis across departments and schemes
- KPI cards for at-a-glance fiscal health monitoring
